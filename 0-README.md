## 内容类型

## 卡片分类

- 反常识卡：`REV`-reverse
- 术语卡：`TER`-terminology
- 人名卡：`PER`-person
- 金句卡：`STC`-sentence
- 行动卡：`ACT`-action
- 技巧卡：`TIP`-tips
- 任意卡：`FRE`-free
- 灵感卡：`INS`-inspiration
- 读书笔记卡：`BOK`-book
- 文章笔记卡：`ART`-artical
- 论文笔记卡：`PAP`-paper
- 演讲笔记卡：`SPC`-speech


## 文件名编码

卡片类别编码（3位数大写字母）+4位数编码+自定义编码（大写字母，位数可自由扩展）
